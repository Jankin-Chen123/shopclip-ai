# Automation Rules

记录以后无需用户重复提醒也要执行的流程。

## Active Rules

### Active Work Disclosure

Only key workflow moments need an active work disclosure. Ordinary short answers, acknowledgements, and simple clarifications can omit the status block.

Use the disclosure block when starting a new stage, switching Agent role, invoking important skills/plugins, implementing code, verifying behavior, deploying, upgrading workflow capabilities, writing key artifacts, or delivering final project output.

```text
当前 Agent：<agent-name>
当前工作：<what is being done>
使用能力：<skills/plugins and why>
当前产物：<file or artifact path, if any>
```
