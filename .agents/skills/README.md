# Skills

本目录保存从 OpenAI skills curated 市场安装的项目开发技能。每个 skill 位于独立文件夹中，并通过 `SKILL.md` 描述适用场景、步骤、输出和常见错误。

## Available Skills

- `figma-use`：读取和使用 Figma 设计上下文。
- `figma-generate-design`：生成或辅助设计 Figma 文件。
- `figma-implement-design`：把 Figma 设计实现为代码。
- `ui-ux-pro-max`：第三方 UI/UX 设计智能 skill，覆盖样式、配色、字体、产品类型、UX 指南、图表、响应式和可访问性建议。
- `playwright`：自动化真实浏览器做端到端检查。
- `playwright-interactive`：交互式浏览器调试和流程验证。
- `screenshot`：截屏、视觉检查和 UI 证据采集。
- `security-best-practices`：安全最佳实践检查。
- `security-threat-model`：威胁建模。
- `security-ownership-map`：安全责任边界梳理。
- `vercel-deploy`：Vercel 部署。
- `netlify-deploy`：Netlify 部署。
- `cloudflare-deploy`：Cloudflare 部署。
- `render-deploy`：Render 部署。
- `openai-docs`：OpenAI 官方文档与 API 使用。
- `jupyter-notebook`：Jupyter Notebook 数据分析和实验。
- `pdf`：PDF 读取、处理和验证。

## Usage Rule

先使用能描述当前阶段的 skill；如果任务跨阶段，按 `AGENTS.md` 的默认项目流程推进。
